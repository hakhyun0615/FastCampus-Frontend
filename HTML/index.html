# 문서의 HTML 버전을 지정(현재 HTML5가 표준) # 문서의 전체 범위(HTML 문서가 어디에서 시작하고, 어디에서 끝나는지 알려주는 역할) # 문서의 정보를 나타내는 범위(웹 브라우저가 해석해야 할 웹 페이지의 제목, 설명, 사용할 파일 위치, 스타일(CSS)같은 웹페이지의 보이지 않은 정보를 작성하는 범위) # meta: HTML 문서의 제작자. 내용, 키워드 같은, 여러 정보를 검색엔진이나 브라우저에게 제공 # 문자인코딩 방식(출력 방식)(UTF-8이 표준) # 제작자 명시 # 뷰포트 속성 명시 # title: (탭에서 볼 수 있는)웹 페이지의 제목 Document # link: 외부 문서(대부분 css)를 가져와 연결할 때 사용 # rel: 관계, href: 가져올 문서의 경로 # style: 스타일(CSS)를 HTML 문서 안에서 작성하는 경우에 사용 # HTML 내부에서 작성된 CSS(권장X, 앞서 언급한대로 파일로 분류해서 가져와서 사용) # script: 자바스크립트(JS) 파일을 가져오거나 HTML 문서 안에서 작성하는 경우 # JS파일을 가져오는 경우 # JS를 HTML 문서 안에서 작성하는 경우 # 문서의 구조를 나타내는 범위(사용자 화면을 통해 보여지는 로고, 헤더, 푸터, 내비게이션, 메뉴, 버튼, 이미지 같은, 웹페이지의 보여지는 구조를 작성하는 범위) # div(division): 특별한 의미가 없는 구분을 위한 블록요소
# h1(heading): 제목을 의미하는 블록요소(숫자가 작을수록 더 중요한 제목을 의미)
